## 🎯 **How to Use Kurdish Customization**

### **Step 1: Open the Kurdish Customizer**
1. **Navigate to:** `kurdish-customizer.html` in your motel folder
2. **Double-click** to open in your browser
3. **See all Kurdish words** used in your system

### **Step 2: Browse Available Words**
The system shows **100+ Kurdish words** organized by categories:

#### **📂 Categories Available:**
- **🧭 Navigation:** داشبۆرد، میوانەکان، خەرجییەکان، ڕاپۆرتەکان
- **📝 Forms:** ناو، ژوور، بەروار، بڕی پارە، ناوی میوان
- **⚡ Actions:** زیادکردن، دەستکاریکردن، سڕینەوە، هەناردن
- **📊 Status:** پارەدراو، چاوەڕوان، بەشی
- **⚙️ System:** سیستەمی بەڕێوەبردنی مۆتێل، ڕێکخستنەکان

### **Step 3: Customize Words**
1. **Find the word** you want to change
2. **See the original Kurdish** in the gray box
3. **Type your preferred translation** in the white box
4. **Changes save automatically** ✅

### **Step 4: Apply to Main System**
1. **Click "💾 Save All Customizations"**
2. **Go back to your main system** (working-motel.html)
3. **Switch language to Kurdish** 
4. **See your custom words** applied instantly! 🎉

## 🎯 **Step-by-Step Customization Tutorial**

### **🔴 Example: Change "میوانەکان" to "میوانان"**

1. **Open Kurdish Customizer:**
   - Double-click `kurdish-customizer.html`

2. **Find the Word:**
   - Type "میوانەکان" in search box
   - Or click "Navigation" category
   - Find the card with "میوانەکان"

3. **Customize:**
   - See "Original Kurdish: میوانەکان"
   - In "Your Custom Kurdish" field, type: "میوانان"
   - ✅ Automatically saved!

4. **Apply to System:**
   - Click "💾 Save All Customizations"
   - Go to main system (working-motel.html)
   - Switch language to Kurdish
   - See "میوانان" instead of "میوانەکان"! 🎉

### **🟡 Example: Customize Multiple Status Words**

1. **Search for Status:**
   - Type "status" in search box
   - Or click "Status" category

2. **Customize All Status Words:**
   - پارەدراو → پارە دراوە
   - چاوەڕوان → چاوەڕێ  
   - بەشی → بەشێک

3. **See Changes:**
   - All payment statuses now use your preferred words
   - Applied throughout the entire system

### **🟢 Example: Create Regional Dialect**

1. **Systematic Changes:**
   - All "ەکان" endings → "ان" endings
   - All "ی" connectors → "ە" connectors
   - All formal terms → colloquial terms

2. **Export Your Dialect:**
   - Click "📤 Export Customizations"
   - Save as "sorani-dialect.json"
   - Share with others using same dialect
