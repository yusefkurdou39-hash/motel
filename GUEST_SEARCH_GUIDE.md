# 🔍 Guest Search System - Complete Guide

## 🎉 **Amazing New Guest Search Feature!**

Your motel management system now has a **powerful guest search system** that allows you to quickly find guests by name, ID/passport, room number, or phone number!

---

## 🚀 **What's New**

### **✅ Comprehensive Search:**
- **Search by Name:** Find guests by first name, last name, or full name
- **Search by ID/Passport:** Quickly locate guests using their identification
- **Search by Room Number:** Find who's staying in specific rooms
- **Search by Phone:** Look up guests by their contact number

### **✅ Smart Features:**
- **Real-time search** - Results update as you type
- **Case-insensitive** - Works with uppercase or lowercase
- **Partial matching** - Find guests with incomplete information
- **Multi-language support** - Works in English, Arabic, and Kurdish
- **Search results counter** - Shows how many guests found

---

## 🎯 **How to Use Guest Search**

### **Step 1: Navigate to Guests Section**
1. **Login to your system**
2. **Click "👥 Guests"** in the navigation menu
3. **See the new search box** at the top of the page

### **Step 2: Search for Guests**
1. **Type in the search box:**
   - Guest name: "John Smith"
   - ID/Passport: "A123456789"
   - Room number: "101"
   - Phone: "555-1234"

2. **See instant results:**
   - Table updates automatically
   - Search counter shows results
   - Highlighted ID/Passport numbers

### **Step 3: Clear Search**
1. **Delete text** from search box
2. **All guests appear** again
3. **Search counter disappears**

```

```

#### **ID/Passport Patterns:**
```
✅ "A123" → All IDs starting with A123
✅ "passport" → All passport entries
✅ "2024" → All IDs/passports from 2024
```

#### **Room Number Patterns:**
```
✅ "1" → Rooms 101, 201, 301, etc.
✅ "01" → Rooms 101, 201, 301, etc.
✅ "suite" → All suite rooms
```

#### **Phone Number Patterns:**
```
✅ "555" → All numbers with 555
✅ "+1" → All US/Canada numbers
✅ "1234" → Numbers ending in 1234
```

### **🚀 Speed Search Shortcuts:**
- **Type first few letters** of name for quick results
- **Use room number** for instant room lookup
- **Search by area code** to find local guests
- **Use passport prefix** for international guests
