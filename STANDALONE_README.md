# 🏨 Standalone Motel Management System

## 📱 **Double-Click to Run - No Server Required!**

This is a completely standalone HTML application that runs directly in your browser without any server or internet connection.

---

## 🚀 **How to Use**

### **Step 1: Open the Application**
- **Double-click** `standalone-index.html`
- Your default browser will open the application
- **That's it!** No server, no installation, no setup required

### **Step 2: Login**
- **Username**: `admin`
- **Password**: `admin123`
- Click "Sign In"

---

## 💾 **Data Storage**

### **Where Your Data is Stored**
- All data is stored in your browser's **localStorage**
- Data persists between sessions
- Data is stored locally on your computer only

### **Backup Your Data**
1. Open browser **Developer Tools** (F12)
2. Go to **Application** tab → **Local Storage**
3. Copy the data or export it
4. **OR** use browser's export/import bookmarks feature

### **Clear Data**
- Clear browser data for this page
- Or delete localStorage entries manually

## 🛠️ **Troubleshooting**

### **Application won't load**
- Make sure JavaScript is enabled in your browser
- Try a different browser
- Check if file is corrupted

### **Data disappeared**
- Check if browser data was cleared
- Look in browser's localStorage
- Restore from backup if available

### **Login not working**
- Use exactly: `admin` / `admin123`
- Check if caps lock is on
- Try refreshing the page

### **Styling looks broken**
- Refresh the page (Ctrl+F5)
- Try a different browser
- Check if CSS is loading properly
