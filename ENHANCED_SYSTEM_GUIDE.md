## 📱 **How to Use Each Feature**

### **🏠 Dashboard**
1. **Login** with any user account
2. **Set date range** using From/To date filters
3. **View statistics** that update automatically
4. **Export to PDF** (if you have export permission)
5. **Export data** as JSON file

### **👥 Guest Management**
1. **Click "Add Guest"** (if you have add permission)
2. **Fill the form:**
   - Enter guest name and ID/Passport
   - Set room number and phone
   - Choose check-in date
   - Enter number of days
   - Set price per day
   - **Watch total calculate automatically**
   - Choose payment status
3. **Save** - Guest is added to the system
4. **View all guests** in the table
5. **Delete guests** (if you have delete permission)

### **🧾 Expense Management**
1. **Add expenses** with categories
2. **Track spending** by date
3. **View expense reports**
4. **Delete expenses** (with permission)

### **📊 Reports**
1. **Export guest data** (JSON format)
2. **Export expense data** (JSON format)
3. **Create system backup** (Admin only)
4. **Restore from backup** (Admin only)

### **⚙️ Settings** (Admin Only)
1. **Change language** - Updates entire system
2. **Set currency** - Changes display currency
3. **Date format** - Choose preferred date format
4. **View users** - See all system users
5. **Create backup** - Full system backup
6. **Restore backup** - Restore from backup file
