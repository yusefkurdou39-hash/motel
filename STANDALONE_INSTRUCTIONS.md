# 🏨 Motel Management System - Standalone Version

## 🚀 Quick Start

### **How to Use:**
1. **Double-click** `working-motel.html`
2. **Login** with:
   - Username: `admin`
   - Password: `admin123`

## ✨ New Features Added

### **Enhanced Guest Management:**
- ✅ **ID/Passport Number** - Required field for guest identification
- ✅ **Phone Number** - Contact information for guests
- ✅ **Days Calculator** - Enter number of days staying
- ✅ **Price per Day** - Set daily rate
- ✅ **Automatic Total** - Calculates total amount automatically (Days × Price per Day)
- ✅ **Payment Status** - Track payment status (Paid/Pending/Partial)

### **Smart Calculations:**
- When you enter **Days** and **Price per Day**, the **Total Amount** is calculated automatically
- No need to manually calculate totals anymore!

### **Enhanced Table View:**
- Shows all guest information in organized columns
- Color-coded payment status badges
- Better organization of guest data

## 📋 Guest Form Fields

1. **Guest Name** - Full name of the guest
2. **ID/Passport** - Identification document number
3. **Room Number** - Assigned room
4. **Phone Number** - Contact number (optional)
5. **Check In Date** - Arrival date
6. **Check Out Date** - Departure date (optional)
7. **Number of Days** - How many days staying
8. **Price per Day** - Daily rate in dollars
9. **Total Amount** - Automatically calculated (readonly)
10. **Payment Status** - Paid/Pending/Partial

## 🎯 How Automatic Calculation Works

1. Enter **Number of Days** (e.g., 3)
2. Enter **Price per Day** (e.g., $50.00)
3. **Total Amount** automatically shows: $150.00
4. Change either field and total updates instantly!
